## 🚀 Getting Started

### 1️⃣ Clone the Repository
```bash
git clone https://github.com/At-an/-Cookie_Demo.git
cd Cookie_Demo
```

### 2️⃣ Make Changes & Commit
```bash
git add .
git commit -m "Your commit message"
git push origin main
```

### 3️⃣ Pull Latest Changes
```bash
git pull origin main
```
